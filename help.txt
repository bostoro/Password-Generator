View passwords - py main.py --passwords
Generate password - py main.py --generate [options]

OPTIONS (not required):
Specify length (minimum {MIN_LENGTH}): --length [value]
Include special characters: --specialChars
Include digits: --digits
Include upper case letters: --upper
Include all possible strengthening options: --mixed

Example usage of --generate commanmd:
py main.py --generate --length 12 --specialChars --upper
py main.py --generate --mixed
